## How It Works

Aplikasi ini menghitung kombinasi cuti paling efisien berdasarkan:
- Libur nasional Indonesia
- Cuti bersama
- Weekend (Sabtu & Minggu)
- Long weekend opportunities

Algoritma akan mencari:
- H-1 / H+1 libur nasional yang nyambung weekend
- Bridge days antara libur dan weekend
- Kombinasi 1-3 hari cuti untuk hasil maksimal
